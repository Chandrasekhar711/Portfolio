# Premium Portfolio Website

This project is a premium interactive portfolio for Bala Venkata Chandra Sekhar Pampana built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, GSAP, Lenis, and React Three Fiber.

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start the development server
- `npm run build` — create a production build

## Highlights

- Hero section with animated typing effect and 3D visual
- Smooth scrolling and scroll progress indicator
- Glassmorphism cards and premium gradients
- Skills, education, project, internship, certification, and contact sections
- Dark/light mode toggle and custom cursor behavior
